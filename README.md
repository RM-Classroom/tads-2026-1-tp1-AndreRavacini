# Sistema de Locadora de Veículos | ASP.NET Core API

![C#](https://img.shields.io/badge/C%23-Programação-blue)
![ASP.NET Core](https://img.shields.io/badge/ASP.NET-Core-purple)
![SQL Server](https://img.shields.io/badge/SQL-Server-red)

API desenvolvida como parte do Trabalho Prático da disciplina, com o objetivo de simular o funcionamento de uma locadora de veículos, aplicando conceitos de modelagem de banco de dados e desenvolvimento backend.

O sistema permite o gerenciamento de entidades como clientes, veículos, fabricantes e aluguéis, com persistência de dados em banco SQL Server utilizando Entity Framework Core.

---

## 🛠 Tecnologias Utilizadas

- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server Express
- Swagger (OpenAPI)

---

## 📌 Funcionalidades (Etapa 1)

- Modelagem conceitual do banco de dados
- Criação do modelo relacional
- Implementação das entidades no backend
- Configuração do Entity Framework
- Geração do banco de dados via migrations
